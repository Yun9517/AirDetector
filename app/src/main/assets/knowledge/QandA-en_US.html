3. Please make sure the device has enough power.

1. If the red light flashes, it means the battery of the device is weak. The device might be shut down in few minutes. In this situation, the device needs to be charged.
2. When the orange light is bright, it means the device is charging. If not, please pull the wire up and then plug in to make sure the device is charging.
3. It takes about 2 hours to get the battery fully charged, and the indicator light will be turned off. At this time, you can remove the device from the power socket.

        Some of the Android phone can not run in background. The reason is that Android System forcibly closes APP in background for saving the battery.
        Due to the APP has already been shut down, it is no longer to support notification and alert.

        TVOC NOSE can work without APP and start detecting while the device is opening.
        All the data will be stored in device and the maximum number of data is 1440.
        Nothing can disrupt detecting even the APP is disconnected due to hibernate. When the APP is connecting to TVOC NOSE by Bluetooth, it will retrieve all the detected result and save in the phone. However, it will lose all the data if you reset the device.
