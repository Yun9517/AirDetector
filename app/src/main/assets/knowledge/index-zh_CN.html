英文 Volatile Organic Compound,三个词第一个字母的缩写，简称VOCs。总挥发性有机物有时也用TVOC来表示 (Total Volatile Organic Compound)。
        按照世界卫生组织的定义，如果在气压101.32kPa下，该化合物的沸点在50℃-250℃，就是挥发性有机物。它们会在常温下以气体形式存在。按其化学结构的不同，可以进一步分为八类：烷类、芳烃类、烯类、卤代烃类、酯类、醛类、酮类和其他。VOC的主要成分有：烃类、卤代烃、氧烃和氮烃，它包括：苯系物、有机氯化物、氟里昂系列、有机酮、胺、醇、醚、酯、酸和石油烃化合物等。以PPB(十亿分之一)为标准单位。

        在室外，主要来自燃料燃烧和交通运输产生的工业废气、汽车尾气、光化学污染等；而在室内则主要来自燃煤和天然气等燃烧产物、吸烟、采暖和烹调等的烟雾，建筑和装饰材料、家具、 家用电器、清洁剂和人体本身的排放等。
        在室内装饰过程中，挥发性有机物主要来自油漆、涂料和胶粘剂。一般油漆中VOC含量为每公升30至70克。由于挥发性有机物具有强挥发性，一般情况下，油漆施工后的10小时内，可挥发出90%。

        挥发性有机物的危害很明显，当居室中挥发性有机物浓度超过一定浓度时，在短时间内人们感到头痛、恶心、呕吐、四肢乏力；严重时会抽搐、昏迷、记忆力减退。
        挥发性有机物伤害人的肝脏、肾脏、大脑和神经系统，其中还包含了很多致癌物质。室内空气被挥发性有机物污染已引起各国重视。

        根据德国联邦环境局所提出的报告而界定, 不同的浓度区间对应不同的暴露时间。请参考以下范围界定及暴露时间建议
        
TVOC的范围界定 (污染程度 / 暴露时间)
        
TVOC < 220 ppb : 正常 / 无限制
TVOC >= 220 ppb (<660): 中等 / 小于12个月内
TVOC >= 660 ppb (<2200): 高 / 小于1个月内
TVOC >= 2200 ppb (<5500): 不健康 / 数小时内
TVOC >= 5500 ppb (<20000): 有害健康
TVOC >= 20000 ppb: 危害健康

        Particulate Matter的缩写，是颗粒物的意思。PM2.5的标准，是由美国在1997年提出的，主要是为了更有效地监测随着工业化日益发达而出现的、在旧标准中被忽略的对人体有害的细小颗粒物。。PM2.5指数已经成为一个重要的测控空气污染程度的指数。

        PM2.5主要对呼吸系统和心血管系统造成伤害，包括呼吸道受刺激、咳嗽、呼吸困难、降低肺功能、加重哮喘、导致慢性支气管炎、心律失常、非致命性的心脏病、心肺病患者的过早死。
        世界卫生组织建议PM2.5之24小时平均暴露浓度不要超过25微克/立方公尺。

<15 : 正常
>=15, <35 : 中等
>=35, <54 : 高
>=54, <150 : 不健康
>=150, <250 : 有害健康
>=250 : 危害健康

        与PM2.5一样，都是属于可吸入颗粒物，区别是颗粒物大小不同。 PM10是指小于等于10微米的可吸入空气悬浮颗粒物。PM2.5是指小于等于2.5微米的可吸入空气悬浮颗粒物。

        PM10对人体的影响与PM2.5相近，但没PM2.5严重。相较于PM2.5，PM10因为颗粒较大，较难入侵人体深层，根据研究，咽喉表面的黏液与鼻腔内部的绒毛皆能阻挡PM10继续入侵人体，但仍有一定程度的危害，如慢性支气管炎与慢性肺炎。
        世界卫生组织建议PM10之24小时平均暴露浓度不要超过50微克/立方公尺。

<54: 正常
>=54, <125: 中等
>=125, <254: 高
>=254, <354: 不健康
>=354, <424: 有害健康
>=425: 危害健康

        eCO2的输出数值是基于空气中的氢气含量，结合其他条件推测出来的，并不是直接测量获得的，不建议拿来与一般的二氧化碳做比较。

        指空气中，实际水蒸气的分压与相同温度和体积下水饱和蒸气压的百分比。用于气象预测之中，它反映了降雨、有雾…等的可能性。
        高的相对湿度会让人感到更热，因为汗水的挥发较慢，也容易不舒服、滋生霉菌、尘螨，引发过敏。湿度较低，太干燥。皮肤也容易过敏且易口渴，要适时补充水分。
        一般来说，湿度维持在45%~65％，不仅凉爽舒适，也不会因为过度干燥或潮湿而影响健康。
