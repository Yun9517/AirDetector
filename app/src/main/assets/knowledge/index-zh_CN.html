英文 Volatile Organic Compound,三个词第一个字母的缩写，简称VOCs。总挥发性有机物有时也用TVOC来表示 (Total Volatile Organic Compound)。
        按照世界卫生组织的定义，如果在气压101.32kPa下，该化合物的沸点在50℃-250℃，就是挥发性有机物。它们会在常温下以气体形式存在。按其化学结构的不同，可以进一步分为八类：烷类、芳烃类、烯类、卤代烃类、酯类、醛类、酮类和其他。VOC的主要成分有：烃类、卤代烃、氧烃和氮烃，它包括：苯系物、有机氯化物、氟里昂系列、有机酮、胺、醇、醚、酯、酸和石油烃化合物等。以PPB(十亿分之一)为标准单位。

        在室外，主要来自燃料燃烧和交通运输产生的工业废气、汽车尾气、光化学污染等；而在室内则主要来自燃煤和天然气等燃烧产物、吸烟、采暖和烹调等的烟雾，建筑和装饰材料、家具、 家用电器、清洁剂和人体本身的排放等。
        在室内装饰过程中，挥发性有机物主要来自油漆、涂料和胶粘剂。一般油漆中VOC含量为每公升30至70克。
        由于挥发性有机物具有强挥发性，一般情况下，油漆施工后的10小时内，可挥发出90%。

        挥发性有机物的危害很明显，当居室中挥发性有机物浓度超过一定浓度时，在短时间内人们感到头痛、噁心、呕吐、四肢乏力；严重时会抽搐、昏迷、记忆力减退。
        挥发性有机物伤害人的肝脏、肾脏、大脑和神经系统，其中还包含了很多致癌物质。室内空气被挥发性有机物污染已引起各国重视。

        根据德国联邦环境局所提出的报告而界定, 不同的浓度区间对应不同的暴露时间。请参考以下范围界定及暴露时间建议
TVOC的范围界定 (污染程度 / 暴露时间)
TVOC < 220 ppb : 正常 / 无限制
TVOC >= 220 ppb (<660): 轻度污染 / 小于12个月内
TVOC >= 660 ppb (<2200): 中度污染 / 小于1个月内
TVOC >= 2200 ppb (<5500): 重度污染 / 数小时内
TVOC >= 5500 ppb (<20000): 严重污染
TVOC >= 20000 ppb: 非常严重污染

        Particulate Matter的缩写，是颗粒物的意思。PM2.5的标准，是由美国在1997年提出的，主要是为了更有效地监测随着工业化日益发达而出现的、在旧标准中被忽略的对人体有害的细小颗粒物。。PM2.5指数已经成为一个重要的测控空气污染程度的指数。

        PM2.5主要对呼吸系统和心血管系统造成伤害，包括呼吸道受刺激、咳嗽、呼吸困难、降低肺功能、加重哮喘、导致慢性支气管炎、心律失常、非致命性的心脏病、心肺病患者的过早死。

<15 : 正常
>=15, <35 : 浓度中等
>=35, <54 : 浓度过高
>=54, <150 : 有害健康
>=150, <250 : 非常有害健康
>=250 : 极度危害健康

        我们的产品测出的等效二氧化碳数值是根据传感器量测再演算而产生的。二氧化碳是空气中常见的化合物, 由两个氧原子与一个碳原子通过共价键连接而成。空气中有微量的二氧化碳，约占400ppm。二氧化碳常压下为无色、无味、不助燃、不可燃的气体。

        大气中二氧化碳的含量在400ppm以上 ，在清净的室内环境中，二氧化碳浓度会接近大气中之浓度。办公室内二氧化碳的来源主要来自于人类呼吸、吸烟、及其他燃烧行为。当室内人员密度过高或是换气效率不佳时，容易造成二氧化碳浓度累积，同时其他的污染物浓度也相对地提高。

eCO2 >= 400 ppm (<700): 正常
eCO2 >= 700 ppm (<1000): 浓度中等
eCO2 >= 1000 ppm (<1500): 浓度过高
eCO2 >= 1500 ppm (<2500): 有害健康
eCO2 >= 2500 ppm (<5000): 非常有害健康
eCO2 >= 5000 ppm : 极度危害健康

        用于气象预测之中，它反映了降雨、有雾…等的可能性。
        高的相对湿度会让人感到更热，因为汗水的挥发较慢，也容易不舒服、滋生霉菌、尘螨，引发过敏。
        湿度较低，太干燥。皮肤也容易过敏且易口渴，要适时补充水分。
        一般来说，湿度维持在45%~65％，不仅凉爽舒适，也不会因为过度干燥或潮湿而影响健康。
